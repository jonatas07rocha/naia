# NAIA - Inteligência Artificial de Alta Performance

A **NAIA** (Núcleo de Analítica e Inteligência Artificial) é uma interface de chat minimalista, rápida e elegante, projetada especificamente para interagir com o modelo **Llama 3.3 70B** através da infraestrutura de ponta da **SambaNova Cloud**.

O projeto prioriza o rigor técnico, a clareza de ideias e a usabilidade em ambientes móveis e desktop.

---

## 🌟 Funcionalidades Principais

* ⚡ **Inferência Ultra Rápida**: Aproveita os chips RDU da SambaNova para entregar centenas de tokens por segundo.
* 🌗 **Suporte a Temas**: Interface com Modos Claro e Escuro (Light/Dark).
* 🎨 **Paleta Mostarda**: Design focado no conteúdo, reduzindo a fadiga visual.
* 📝 **Renderização Markdown**: Suporte total para textos estruturados e blocos de código.
* 📊 **Monitor de Quota**: Sistema local que alerta sobre o limite diário de mensagens do plano gratuito.
* 🔒 **Privacidade Local**: Suas chaves de API e preferências residem apenas no seu navegador (`localStorage`).
* 📱 **Mobile Ready**: Design responsivo otimizado para Safari (iPhone) e navegadores Android.

---

## 🚀 Como Executar

1.  Salve o arquivo `index.html` em uma pasta.
2.  Abra o arquivo no navegador.
3.  Obtenha sua chave de API em [SambaNova Cloud](https://cloud.sambanova.ai/).
4.  Configure a chave no ícone de engrenagem da aplicação.

---

## 🛠️ Tecnologias Utilizadas

* **JavaScript Vanilla**: Lógica pura sem dependências pesadas.
* **CSS3 Custom Properties**: Gerenciamento dinâmico de temas.
* **Lucide Icons**: Iconografia minimalista.
* **Marked.js**: Renderização eficiente de Markdown.
